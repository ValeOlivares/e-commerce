# E-commerce app

E-Commerce web app based React

## Run Locally

### 1. Clone repo

```
$ git clone git@github.com:ValeOlivares/e-commerce.git
$ cd e-commerce
```

### 2. Run locally

```
$ npm install
$ npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
